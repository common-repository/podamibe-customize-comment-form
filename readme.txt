== Description ==

<strong>Podamibe Customize Comment Form </strong> is a <strong>FREE WordPress plugin </strong> which help you to display custom fields on WordPress comment form like phone number, country, title, rating and also change the default textarea to editor.

== Free Features ==

* <strong>User can book date from frontend also</strong>
* <strong>Five Custom fields</strong>
* <strong>Multilanguage Translation Ready</strong>
* <strong>Easy to use</strong>

== Installation ==

1. Upload the plugin 'Podamibe Customize Comment Form' to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings -> Customize Comment Form for detail settings.


== Frequently Asked Questions ==

= What are Form setttings ? =

There are the fields list to display on comment form.

= How to display custom fields ? =

Just check Yes to diplay that field on comment form

= How to hide custom fields ? =
Just check No to hide that field on comment form
